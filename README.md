# **100 Exercises for Practicing Python**

This repository documents my personal journey through **_100 Exercises for Practicing Python_** by **Laurentine K. Masson** — a collection of programming challenges organized into three levels of complexity:

- 🟢 **Beginner**
- 🟡 **Intermediate**
- 🔴 **Advanced**

For each exercise, I follow a consistent process:

1. ✍️ I write down the exercise question **exactly as the author presents it**
2. 🛠️ If the book contains a typo or mistake, I **fix it or point it out**
3. 🧠 I solve the problem in Python
4. 📝 I add notes when helpful — patterns, mistakes, or alternative approaches

This project is focused on learning, practicing, and improving through repetition and exploration.

---

## 📁 **Project Structure**

```
100_Exercises_practicing_python/
│
├── Beginner_Exercises/
├── Intermediate_Exercises/
├── Advanced_Exercises/
│
├── README.md
└── pyproject.toml
```

Each exercise folder may include:

- `question.md` — the exercise prompt written exactly as given in the book (with corrections noted if needed)
- `exercise.py` — my solution
- `notes.md` — optional thoughts or explanations

---

## 🧱 **Environment Setup (UV)**

This project uses **UV** for a clean, reproducible Python environment.

Create or activate the environment:

```bash
uv init
uv venv
uv pip install --upgrade pip
```

Run any exercise:

```bash
uv run python Beginner_Exercises/ex01/exercise.py
```
